# **App Name**: BCC Meeting Management

## Core Features:

- Dashboard: Display upcoming meetings, quick actions, and a meeting list in a 'Bento Box' grid layout.
- Meeting Detail & Live Mode: Show agenda, attendees, and a presentation mode (slide deck) in a split view during live meetings.
- Interactive Team Grid/Carousel: Render the 'Pembagian Tugas' slide as an interactive team grid or carousel.
- Minutes of Meeting (MoM) Input: Provide a rich-text input area for inputting meeting minutes.
- AI-Powered MoM Generation: Enable automated generation of meeting minutes with AI tool, from meeting audio and/or video.
- Task Management Integration: Link action items from the MoM section directly to a task management system for tracking.

## Style Guidelines:

- Primary color: Red (#CA1F3D) for FABs, primary buttons, and active states.
- Secondary/Accent color: Yellow (#FFBE00) for highlights, badges, and illustrations.
- Background color: Deep charcoal/black (#121212) for a dark mode default to make the Red/Yellow pop.
- Font recommendation: 'Inter' sans-serif for both headlines and body text, due to its suitability for modern interfaces and legibility. Note: currently only Google Fonts are supported.
- Utilize a 'Bento Box' grid layout for the dashboard, enhancing visual appeal and content organization.
- Implement a navigation rail on the left side for primary navigation.
- Incorporate cards with elevation to create visual depth and hierarchy.
- Use fluid animations (e.g., Framer Motion) and expressive micro-interactions for a dynamic user experience.
- Incorporate glassmorphism effects on cards to add a modern aesthetic.
- Employ Lucide React icons for consistent and visually appealing icons throughout the app.
